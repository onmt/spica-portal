ffkit · 映像納品の道具

完成した動画を、
現場で確実に再生できる形に。

編集の終わった動画を、プレーヤーに挿せばそのまま再生が始まる DVD / BD-Video や、配布用の MP4 にします。何十本もある案件は、まとめて自動で。オフラインの CLI なので、映像が端末の外に出ることはありません。

編集が終わってからが、地味に手間でした。プレーヤーに挿したら、メニューもなく、すぐ本編が始まってほしい。何十本もある案件は、できれば寝ている間に片付けたい。

汎用の変換ソフトは「変換」まではやってくれます。でも、現場で確実に再生できる納品物にするところまでは、面倒を見てくれませんでした。

ffkit は、その最後のひと手間だけを引き受けます。

できること

挿せば、再生が始まる。

メニューなし DVD / BD-Video

挿すと、すぐ本編が始まります。視聴者が操作に迷うメニューは、作りません。

AI からも使える

なぜ ffkit か

現場で使いながら、作りました。

自分の納品で困った

汎用ツールだと取りこぼしてしまう細かいところを、自分の案件で焼いて、確かめながら直してきました。

ローカルで完結

素材も書き出したファイルも、アップロードしません。ぜんぶ、あなたの端末の中だけで終わります。

ライセンスと安全（正直に）

できないことは、
できないと書きます。

コア＝無料（PyPI / GitHub）。商用ライセンスは予定です（価格未定）。

焼いたディスクは、お手元のプレーヤーで一度、再生を確かめてください。 ffkit は AS-IS・無保証で出しています（EULA）。

特許や GPL の問題は、根元で避けています。 ffmpeg などの外部ツールは同梱せず、別のプロセスとして呼ぶだけ。依存は THIRD_PARTY に書いて、ライセンスを守ります。

プライバシー：収集しません（オフライン）。プライバシー方針

入手

使ってみる。

PyPI から入れるか、GitHub から取得してください。技術者向けの CLI / desktop ツールです。

# PyPI から
pip install ffkit

# 変換して、無人で回す
ffkit build ./input --target dvd --out ./out
GitHub で見る → ドキュメント
For developers worldwide. ffkit turns finished videos into menu-free DVD-Video / BD-Video and distribution MP4 — an unattended, deterministic, resumable batch for the last mile of delivery. Insert and it just plays. Offline CLI; your footage never leaves your machine. Free core (PyPI / GitHub); commercial license planned. Provided as-is — verify your discs play. We don't bundle ffmpeg / GPL binaries (we call system tools); third-party licenses respected.

あなたのデータを、あなたの手元に。スピカ・イメージクリエイション。
